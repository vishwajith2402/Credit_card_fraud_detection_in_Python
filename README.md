# Credit_card_fraud_detection_in_Python
A machine learning-based credit card fraud detection system in Python that allows users to select a dataset via a file manager, analyzes transaction data, and classifies transactions as fraudulent or normal using a Random Forest model, with visualization and performance evaluation.
This project is a Machine Learning-based application developed in Python to detect fraudulent credit card transactions. It allows users to dynamically select a dataset using a file manager and performs data analysis, visualization, and classification using a Random Forest model.
The system processes transaction data, identifies patterns, and classifies each transaction as either legitimate or fraudulent. It also provides performance metrics such as accuracy, confusion matrix, and classification report to evaluate the model.

**Features**
File picker to upload dataset (CSV)
Data preprocessing and validation
Visualization of transaction distribution
Fraud detection using Random Forest algorithm
Model evaluation with detailed metrics
Sample transaction prediction

**Technologies Used**
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Tkinter (for file selection)

**Dataset Requirement**
CSV file format
Must include a Class column:
0 → Normal Transaction
1 → Fraudulent Transaction

**How to Run**
Install required libraries
Run the Python script
Select dataset using file manager
View results and predictions

**Output**
Graphical visualization
Fraud classification results
Model performance metrics
